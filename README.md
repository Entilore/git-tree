gitTree
====

List contents of repository in a tree-like format, using .gitignore to hide files

## Demo

![git-tree.png](https://qiita-image-store.s3.amazonaws.com/0/21173/51cd4acc-e1bc-fe64-203b-6aef08ac8c60.png "git-tree.png")

## Requirement
 - python

## Usage
```
$git-tree
```
or
```
$git tree
```
`
